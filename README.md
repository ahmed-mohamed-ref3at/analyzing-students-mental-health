# Analyzing Students' Mental Health

This repository contains a learning SQL case study based on a DataCamp project.

## Objective
Analyze how the length of stay (stay) affects the mental health of international students.

## Dataset
- Source: DataCamp
- Table: students
- Focus: International students only

## Metrics Analyzed
- Depression score (PHQ-9)
- Social connectedness score (SCS)
- Acculturative stress score (ASISS)

## Skills Used
- SQL
- Data aggregation
- GROUP BY and filtering
- Exploratory data analysis

## Notes
This project is for learning and practice purposes.
